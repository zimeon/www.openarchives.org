ResourceSync Framework Specification - Tools

22 February 2017

This page lists Tools and Additional Resources to support use of the ResourceSync Framework Specifications. Feedback is most welcome on the ResourceSync Google Group.

Schema

  • W3C XML Schema for validating ResourceSync extensions to the Sitemaps document formats

Python

  • ResourceSync libraries in Python and a command-line client: https://github.com/resync/resync
  • ResourceSync notifications including Source, Hub and Destination: https://github.com/resync/resourcesync_push
  • resync-simulator - source simulator in Pyhton: https://github.com/resync/resync-simulator
  • A consumer for arxiv using ResourceSync for the SHARE project - https://github.com/erinspace/arxiv

Ruby

  • resync, a Ruby gem providing an object mapping to and from ResourceSync XML documents: https://github.com/CDLUC3/resync
  • resync-client, a gem that builds on resync to simplify traversing a ResourceSync document tree and downloading resources from it: https://github.com/CDLUC3/resync-client
  • Validate Sitemap and ResourceSync manifests - https://github.com/cbeer/resourcesync-validator

Java

  • Java library - https://github.com/CottageLabs/ResourceSyncJava
  • For DSpace - https://github.com/CottageLabs/DSpaceResourceSync

Scala

  • Scala library - https://github.com/richardrodgers/resync

PHP

  • https://github.com/stuartlewis/resync-php
