We have several components which could be part of a &quot;suspicious paper&quot; detector in small-molecule chemistry (and possibly crystallography). This term is meant to cover at least:<br>* syntactic and stylistic problems. This is not per se an indication that something has been deliberately fudged but it&#39;s an indication of poor quality. This includes incorrect chemical names and locally incorrect english syntax.<br>
* internal inconsistency. Examples are spectral peaks against element count, names not consistent with formula, etc.<br>* inconsistency between values in this paper and the rest of the literature. This can be done for crystallographic structures (e.g. in CrystalEye) or for chemical shifts.<br>
* computation. This relates particularly to what Marlon is doing in OREChem.<br><br>There is a lot of tedious work to do but it&#39;s certainly possible to do some of these on a high-throughput basis.<br><br>I am now fairly bullish about interpreting cvhemical reactions and we canm extract a lot of this from supplemental data.<br>
