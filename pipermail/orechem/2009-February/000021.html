I played around with both environments this morning a bit more and
agree with Jim Downing's evaluation. Sharepoint works great if all of
us are committed to operating within Internet Explorer environment.
However, as soon as you move to another browser or into another
operating system other than Windows its functionality starts to
degrade. For example the wiki, which has a really nice WYSIWYG
interface in Internet Explorer, is almost impossible to use in Firefox
or Safari. The same is true with the revision system and sharing
system for office documents. It works great on Windows office, has no
functionality on Mac office. So, given that a number of us work
outside the Windows domain on both Linux and Macs I think we should
stay with TRAC, which seems to have lesser functionality but is
browser agnostic. Thanks for other peoples comments and help on this.
