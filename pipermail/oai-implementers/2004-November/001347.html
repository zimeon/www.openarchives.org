David,

<description> is used to extend Identify responses.  <about> is used to
describe individual metadata records (i.e., <about> describes the stuff
inside <metadata>).

it sounds like you want to convey information about the repository as a
whole.  if this is true, make a schema for what you would like to say and
wrap your statements inside a <description> block inside an Identify
response.
