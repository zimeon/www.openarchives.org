This is a message to selective harvesters of records for LC digitized
historical content.  Today we released a new set of records -- for about
250 digitized books, corresponding to the American Memory collection
  American Notes: Travels in America, 1750-1920                           
  http://memory.loc.gov/ammem/lhtnhtml/lhtnhome.html                      
                                                                          
We also created a combined set -- lcbooks -- which for now just includes
this collection and the collection of dance manuals released some time
ago.  More books will be added over the coming months.  Some may be
individual books digitized for reasons other than inclusion in an American
Memory collection.
                                                                          
Please let me know if there are any problems.
