I'd like to announce the availability of a thesis and dissertation OAI 1.0
repository with the base URL of
http://purl.org/alcme/etdcat/servlet/OAIHandler. It passed the Repository
Explorer 1.0 examination and has been added to its list of tested archives.
The repository currently contains a sample of about 100,000 thesis and
dissertation records extracted from OCLC's WorldCat database. Documentation
for the repository, such as it is, can be found at
http://purl.org/alcme/etdcat/index.html.

Currently, the repository only supports the oai_dc format. Eventually, it
will also support an enhanced Dublin Core format to be defined by the
Networked Digital Library of Theses and Dissertations organization (NDLTD).

I'd also like to note that this repository was built using OAICat software
that will eventually be available to the OAI community under an open-source
license. The idea for this software is to make it as simple as possible for
people to add OAI protocol capability to an existing database.
Documentation, as it accumulates, can be found at
http://alcme.oclc.org:4342/OAICat.html. If this sounds useful to you,
contact me for details.
