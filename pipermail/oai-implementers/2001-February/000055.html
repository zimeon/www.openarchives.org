Hi all,

Here is another sample implementation of the OAI protocol for anyone who
might be using a Windows NT platform.  You can download the ZIP file from
the following URL:

http://dli.grainger.uiuc.edu/publications/twcole/oaialpha/

This is a simple implementation, intended for illustration only, using the
Microsoft Windows Active Server Pages (ASP) and ActiveX Data Object (ADODB)
technology.  A sample MS Access database is included, but the scripts could
be easily modified to support other data sources such as SQL Server or
others.

See the included README file for minimalist documentation.

Anyone is free to use this however they see fit; all the usual caveats
apply, of course.

Any and all questions, comments, or bug reports are welcome.
