Hello,
 
If this question has already been asked (and answered): sorry for the inconvenience.
 
Q: I would like to use the ListIdentifiers function with more than one metadata format. The way I understand OAI, I have to do multiple requests to get results for more than one metadata format, e.g.:
 
1st request: http://...../handleRequest.xml?verb=ListIdentifiers&metadataPrefix="oai_dc"
2nd request: http://...../handleRequest.xml?verb=ListIdentifiers&metadataPrefix="oai_czp"
 
I already found an issue that covers this question [1]. It seems that the issue is still open, but it is a 2001 issue...!
 
Could somebody tell me if this issue is resolved in OAI version 2.0? 
 
To make it more clear, I would like to do one of the following requests:
 
http://...../handleRequest.xml?verb=ListIdentifiers&metadataPrefix="oai_dc&metadataPrefix="oai_czp"
OR
http://...../handleRequest.xml?verb=ListIdentifiers&metadataPrefix="oai_dc,oai_czp"
 
Thank you in advance,
 
Henk Enting

[1] http://www.dlib.vt.edu/projects/OAI/reports/OAI-Tech_Issue__Multiple_Best_Metadata_Return.html
