Yes, there are. The aforementioned Zebra server from Indexdata (it's
OSS, http://www.indexdata.dk/) comes with example configuration for
indexing OAI-PMH responses. It's done using either the "ALVIS" or "DOM"
filter modules (they're new as of Zebra 2), which will allow you to
customize the generated indexes by means of XSLT.

Don't hesitate to ask further questions, but if it's Zebra-specific,
better ask on the Zebra mailing list.
