<P><FONT SIZE=2>Hi everybody!<BR>
<BR>
The National Library of Sweden has released OAI4J, a client library for OAI-PMH and OAI-ORE written in Java, as Open Source. The project is hosted on Sourceforge and can be found at:<BR>
<BR>
&nbsp;&nbsp; <A HREF="http://oai4j-client.sourceforge.net">http://oai4j-client.sourceforge.net</A><BR>
<BR>
There you can find links to a binary download, Java documentation for the API and some examples to get you started. The source code can also be found and downloaded with the help of Subversion.<BR>
<BR>
OAI4J is released as Open Source under the Apache License, version 2.0.<BR>
<BR>
Features:<BR>
&nbsp;- OAI-PMH<BR>
&nbsp;&nbsp; * Convenient Java API that lets you perform queries and handle the harvested data in<BR>
&nbsp;&nbsp;&nbsp;&nbsp; an object-oriented fashion.<BR>
&nbsp;&nbsp; * Handles all the verbs and responses of OAI-PMH.&nbsp;&nbsp;&nbsp;<BR>
<BR>
&nbsp;- OAI-ORE<BR>
&nbsp;&nbsp; * Lets you create and build Resource Maps from scratch programmatically.<BR>
&nbsp;&nbsp; * Handles parsing of existing Resource Maps written as Atom feeds.<BR>
&nbsp;&nbsp; * Allows Resource Map objects to be serialized into Atom feeds.<BR>
&nbsp;&nbsp; * Could be extended to also handle RDF/XML parsing/serialization.<BR>
<BR>
