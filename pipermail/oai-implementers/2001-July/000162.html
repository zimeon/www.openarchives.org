Thomas Krichel wrote:
>   How does one install a copy of this validator on a local unix
>   box? I think Hussain has done that... I would be grateful for
>   anyone to point me to the right tool to get this done, I can
>   find out details myself.

you have to install LTXML and PyLTXML (from the Language Technology
group at Edinburgh - http://www.ltg.ed.ac.uk/software/xml/index.html)

then you need the source files from the W3C's CVS repository
(http://dev.w3.org/cvsweb/xmlschema/)

and to run the checker you issue something like "python applyschema.py
<xml_file>"

good luck
