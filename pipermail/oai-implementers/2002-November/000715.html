We are nearing the end of the 6-month process of
upgrading from OAI-PMH 1.1 to 2.0.  

As of December 1, 2002, two things will happen: 
we will no longer validate sites that are not 
version 2.0; and currently registered sites that
are still at 1.1 will be removed from the database 
shown at  
http://www.openarchives.org/Register/BrowseSites.pl.

At the same time we will update the tools and 
service provider lists to contain only sites that 
support version 2.0 (they might also support 1.0 
and 1.1).

These changes should not affect most readers of
this list, but we wanted to remind everyone of
this final step in the planned migration to
the current version of the OAI-PMH.
