Hello all,

I'd like to announce a beta release of our open-source OAI server and
harvester applications.

- OAICat (http://www.oclc.org/research/software/oai/cat.shtm) is a Java
Servlet web application providing an OAI-PMH v2.0 repository framework. The
framework can be customized to work with arbitrary data repositories by
implementing some Java interfaces. A demonstration implementation is
available for download on the OAICat home page.

- OAIHarvester (http://www.oclc.org/research/software/oai/harvester.shtm) is
a Java application providing an OAI-PMH v2.0 harvester framework. This
framework can be customized to perform arbitrary operations on harvested
data by implementing some Java interfaces. A demonstration implementation is
available for download on the OAIHarvester home page.

Thanks.
