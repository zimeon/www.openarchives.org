I disagree. Arguments that are not 'required', 'optional' or 'exclusive'
are 'illegal' and should generate badArgument errors. This is how I read
the spec for badArgument:

http://www.openarchives.org/OAI/openarchivesprotocol.html#ErrorConditions
"The request includes illegal arguments, is missing required arguments,
includes a repeated argument, or values for arguments have an illegal
syntax."

Cheers,
Simeon
