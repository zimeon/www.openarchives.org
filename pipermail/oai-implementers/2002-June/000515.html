Apologies for cross-posting, FYI,

There is an OAI-PMH v.2.0-beta compliant data provider for the Library
of Congress OAI repository at (base url):

http://memory.loc.gov/cgi-bin/oai2_0

This new data provider contains an implementation of the new MARC21 xml
schema that Herbert mentioned in a previous email. I used
metadataPrefix=lc_marc21 for this format for now. The functionality of
this data provider has been validated with the openarchives.org registry
validator (though not registered since the repository Id is in use with
our v1.1 implementaion). I also got successful results from the "Test
and Add" link in the Repository Explorer. 

Thanks, please let me know if you find any problems with this
implementation.

Dave Woodward
Library of Congress
