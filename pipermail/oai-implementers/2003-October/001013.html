Hello,

I announce DC Checker
(http://harvest.physik.uni-oldenburg.de/dc/dcchecker.php).

It is a simple tool which checks the content of your metadata records
against recommendations by the Dublin Core Metadata Initiative (DCMI).

Additionally it tries to find wrong UTF-8 encodings (with analysis), and
does some additional checks and presents this in a statistic, so you can
easily see how your Data Provider is doing.

Currently it does quite simple checks, but the recommendations are mostly
not very specific anyway. It does some additional checks to ease life for
Service Providers and will create some warnings then.
(sometimes picky, it will create a warning if creative commons license is
not used...)

Feedback is very welcome, whether these checks make sense, or in what way
this tool can be improved.
