sorry for the late answer ...

DINI (German Inititave for Networked Information) has just published a
new version of recommendations for OAI data providers at German
universities. This paper mainly recommends three types of set
categorization and the appropriate semantic interpretation:

1. Publication Type, e.g. pub-type:dissertation or pub-type:article
2. Document Type, e.g. doc-type:text or doc-type:video
3. Subject Type, e.g. ddc:004 (according to DDC)

The recommendations are available at
http://www.dini.de/documents/2003-10-08-OAI-Empfehlungen.pdf
(Sorry, in German only, but setName is listed in English - so look at
the three tables for the different set types. There is an English
translation of a prior version of these recommendations avaliable at
http://www.dini.de/documents/DINI-OAI-e.pdf, the new version will be
available in English soon.)

Some of the German data providers have introduced these sets into their
OAI interfaces, and there has also been implemented a prototype for a
service provider using these set recommendations which can be found at
http://edoc.hu-berlin.de/e_suche/oai.php
