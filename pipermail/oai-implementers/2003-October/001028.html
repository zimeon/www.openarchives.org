All,

An updated version (2.0.1) of the OAI Static Repository Gateway software created 
by the Digital Library Research & Prototyping Team of the LANL Research Library 
is available at http://sourceforge.net/projects/srepod/

This version is compliant with the recently released beta specification for an
OAI Static Repository and an OAI Static Repository Gateway (see
http://www.openarchives.org/OAI/2.0/guidelines-static-repository.htm)

Your feedback is most welcome.
