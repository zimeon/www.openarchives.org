I would like to announce a new Java OAIHarvester2 package available at
http://www.oclc.org/research/software/oai/harvester2.htm. This package
supports harvesting of both OAI v1.1 and v2.0 repositories.

This is an incompatible reimplementation of an earlier harvester that only
supported OAI v2.0. The old version is still available to support existing
users (http://www.oclc.org/research/software/oai/harvester.htm).

In addition to supporting both OAI-PMH v1.1 and v2.0, the new harvester is
much more lightweight. It can run as a standalone application or can be
integrated into existing applications. Whereas the old application had a
Java API that allowed processing of harvested records on the fly, the new
version simply consolidates the OAI responses into a single XML file for
subsequent processing (e.g. with XSLT).

Comments and suggestions are welcome.

Jeff
