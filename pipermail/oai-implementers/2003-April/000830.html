One of the ways we think about the two different identifiers is this:


The DC identifier is an identifier for the RESOURCE.  In fact, our rule
of thumb is that any information within the metadata tag is about the
RESOURCE.

The OAI identifier is an identifier for METADATA for the resource.  And
again, information in the header tag is about METADATA for the RESOURCE.
We tend to think of the about tag as being about the METADATA as well.
