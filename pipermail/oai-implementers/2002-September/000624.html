  Greetings,

  for work relating to the Academic Metadata Format, I have prepare a 
  script that generates an XML Schema type "mimeType" that contains
  XML schema file to check for correct values of mime types. The
  schema is available at 

http://amf.openlib.org/2001/mime-types.xsd

  I will keep this file up to date at this URL, but if you prefer to
  generate the list yourself, you can get the script from

http://amf.openlib.org/tool/mimetypexsd

  I trust this will holp others writing schema files.

  I will be grateful for reports on any mistakes that I made.
