hi

i didn't scrutinize too much, but here are some things to look at for 
starters:

- when i issued GetRecord, the record that came back had the wrong root 
node - "MTD-BR" as opposed to "Documento.teses" used in the schema

- the Identify response used the wrong schema (and namespace too?) for 
the toolkit portion so it did not validate

- ListIdentifiers did not work because the completeListSize is an empty 
string - if it appears in the XML, i think it has to be a positive integer

- you may need a "targetNamespace" attribute at the top of your schema - 
see the OAI protocol schema for an example of how its done

hope this is useful to you (and to others too)
