Hi everyone,
=20
I'm currently working on a 4th year Computer Science project at the =
University of Cape Town in South Africa, under the supervision of =
Hussein Suleman.
=20
Our project involves investigating the applicability of the SOAP =
messaging framework (version .2) to OAI-PMH.  I am developing a tool =
which enables one to test a repository for conformance to the new =
experimental encoding as well the original encoding of OAI-PMH v2.0.  =
This tool is completely self-contained and runs on your machine as a =
Java application, which should make it more useful in some instances =
than central tools like the Repository Explorer.
=20
I am nearing the end of my work and would appreciate if anyone =
interested would download the tool and try it out.  Any feedback =
(positive or negative) would be greatly appreciated and would help me to =
locate any errors or omissions.

Please note that the experimental encoding has been disabled in this =
version of the tool as it is still under development, and thus only =
OAI-PMH v2.0 can be used.
=20
The tool can be downloaded at the following location:

www.cs.uct.ac.za/~bmerchan/Testtool.zip

=20
Regards,
=20
Bhavik Merchant
University of Cape Town
