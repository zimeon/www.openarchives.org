Hi all. If the Z39.50 resource is able to be queried as a SQL database, 
the VTLS library system vendor have produced a thing called "Vortex". 
It's written for their Virtua library system (which we don't have by the 
way). I've downloaded it and briefly looked at it, and it looks like 
you'd be able to modify it to allow OAI harvesting from a MARC database 
that's query-able via SQL queries. It uses some SQL queries (via Perl) 
to query for sets of records from the Virtua catalogue. You might find 
that you can configure it to meets your needs. It's open source, so 
there's no problem with downloading and using or modifying it.

It's available at:
http://www.vtls.com/Products/vortex.shtml

Simon
