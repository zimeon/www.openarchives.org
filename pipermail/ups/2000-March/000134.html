> 
>   What is the advantage of implementing only the subset versus
>   implementing a full dienst server?

The subset is just a stub front end.  It parses the defined Open arhicives
protocol requests and sends them requests to a stub front end.  The full
release contains the full set of dienst services (repository, indexing,
collection management, user interface).  THe repository portion of the full
release contains modules for submitting documents, storing them int he file
system and generating formats.
