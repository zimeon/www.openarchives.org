Dear all,

We've installed our new Dienst server at Cornell which is now fully
compliant with the latest Dienst protocol at
http://www.cs.cornell.edu/cdlrg/dienst/protocols/DienstProtocol.htm, which
is a superset of the OAMS Dienst protocol at
http://www.cs.cornell.edu/cdlrg/dienst/protocols/OpenArchivesDienst.htm.
For example, the following verbs:

http://cs-tr.cs.cornell.edu/Dienst/Repository/4.0/List-Contents?file-after=2
000-01-01

http://cs-tr.cs.cornell.edu/Dienst/Repository/2.0/List-Partitions

http://cs-tr.cs.cornell.edu/Dienst/Repository/1.0/List-Meta-Formats

http://cs-tr.cs.cornell.edu/Dienst/Repository/2.0/Structure/ncstrl.cornell/T
R95-1540?view=%23

http://cs-tr.cs.cornell.edu/Dienst/Repository/1.0/Disseminate/ncstrl.cornell
/TR2000-1780/%23oams/xml

This new release of DIenst will be available for general distribution in the
near future (a few packaging details.

Also, if you go to the UI page for this server (http://cs-tr.cs.cornell.edu)
you'll see the Open Archives logo displayed (with pride).

Thanks to David Fielding for this work.
