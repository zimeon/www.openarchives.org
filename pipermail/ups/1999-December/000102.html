Hi all,

We've installed a running version of the Open Archives Reference software
with some dummy data.  You can view it through the following URL;

Server Identity:
http://www.ncstrl.org:8070/Dienst/Info/1.0/Identity

Server Services:
http://www.ncstrl.org:8070/Dienst/Info/3.0/List-Services

Simple List-Contents:
http://www.ncstrl.org:8070/Dienst/Repository/4.0/List-Contents

List-Contents with Metadata Dump:
http://www.ncstrl.org:8070/Dienst/Repository/4.0/List-Contents?meta-format=o
ams

Disseminate Metadata for a record:
http://www.ncstrl.org:8070/Dienst/Repository/1.0/Disseminate/xxx/001/%23oams
/xml

List-Partitions:
http://www.ncstrl.org:8070/Dienst/Repository/2.0/List-Partitions

Show metadata available for a record:
http://www.ncstrl.org:8070/Dienst/Repository/1.0/Structure/xxx/001?view=%23

Show available metadata formats at Archive:
http://www.ncstrl.org:8070/Dienst/Repository/1.0/List-Meta-Formats
