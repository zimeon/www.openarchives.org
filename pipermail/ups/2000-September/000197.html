Dear All,

The public beta version of the eprints.org archive software is now
available for download from:

   http://www.eprints.org/

This is a beta release. It is not proven and is not yet suitable for a
live service. It is provided only for testing and evaluation purposes.

Please be sure to use the bug tracking system to report bugs and make
suggestions for improvement:

   http://bugs.eprints.org/

Before you download the beta, please also be sure to read the information
on the "Software Download" page.

Happy archiving!
