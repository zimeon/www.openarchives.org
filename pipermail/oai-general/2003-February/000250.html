IMS is a global consortium that develops open specifications to support
the delivery of e-learning through Learning Management Systems (LMS).

People might like to note that version 1.0 of the IMS Digital Repositories
Specification is available at

 http://www.imsglobal.org/digitalrepositories

It recommends use of the OAI-PMH for harvesting IMS metadata records
between cooperating elearning systems (e.g. from a learning object
repository to a learning management system).

The following article

 Exposing information resources for e-learning - Harvesting and searching
 IMS metadata using the OAI Protocol for Metadata Harvesting and Z39.50
 by Steve Richardson and Andy Powell
 Ariadne issue 34
 http://www.ariadne.ac.uk/issue34/powell/

describes some implementation experience of using the OAI-PMH to exchange
IMS metadata.
