Dear all,

The Digital Library Group in Old Dominion University is pleased to
announce the availability of Arc through SourceForge
(http://sourceforge.net/projects/oaiarc/). Arc is released under the NCSA
Open Source License.  Arc is a federated search service based on the Open
Archives Initiative Protocol for Metadata Harvesting (OAI-PMH).  It
includes a harvester which can harvests both OAI-PMH 1.x and 2.0 compliant
repositories, a search engine together with a simple and advanced search
interface, and an OAI-PMH layer over harvested metadata. It is based on
Java Servlet technology and requires JDK1.4, Tomcat 4.0x, and a RDBMS
server (tested with Oracle and MySQL).

Arc can be configured for a specific community, We encourage the community
to participate in (besides using), extending and enhancing  Arc by
providing support for parallel metada set specific to different
communities, enhancing and customizing search interface, adding post
processing etc. A demonstration instantiation is running at
http://arc.cs.odu.edu, it currently harvests some 100 OAI-PMH compliant
repositories.
