This document is one part of the Implementation Guidelines that accompany the Open Archives Initiative Protocol for Metadata Harvesting (OAI-PMH).

A recommended XML Schema to represent MARC21 records

Since the release of the XML Schema for MARC21 metadata by the Library of Congress, June 2002, usage of MARC21 instead of oai_marc is strongly recommended.

A stylesheet supporting migration from oai_marc to MARC21 is provided on the MARC21 pages of the Library of Congress.

It is recommended that the metadataPrefix "marc21" be used with this metadata format.
